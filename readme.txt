Pasos para compilar SASS con GULP

1. Instalar gulp globalmente: npm install -g gulp
2. Iniciar package.json: npm init
3. Instalar node_modules: npm install
4. Instalar gulp y gulp-sass en las Dev. Dependencias
	- npm install --save-dev gulp
	- npm install --save-dev gulp-sass
5. Hacer gulpfile.js con los task necesarios
